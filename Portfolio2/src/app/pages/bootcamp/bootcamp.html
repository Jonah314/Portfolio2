Week 2: JavaScript Fundamentals

Week 3: Advanced JavaScript I

Explored modern JavaScript (ES6+) features including let/const, arrow functions, constructors, the new keyword, and object-oriented programming with classes. Applied these concepts by building a Deck of Cards application.

Week 4: Advanced JavaScript II

Week 5: Introduction to TypeScript

Studied TypeScript fundamentals, including compiler setup, tsconfig configuration, type inference, primitive and complex types, union types, typed functions, arrays, and object typing.

Week 6: Advanced TypeScript

Expanded TypeScript knowledge through interfaces, type aliases, classes, generics, and advanced type-safe programming concepts that prepared the foundation for Angular development.

Week 7: Angular Fundamentals

Learned Angular architecture, components, templates, data binding, modules, event handling, directives, pipes, lifecycle hooks, and component communication techniques.

Week 8: Modern Angular Syntax

Worked with Angular's modern control flow syntax and compared it to legacy approaches to improve application readability, maintainability, and performance.

Week 9: Intermediate Angular Development

Implemented Angular routing and navigation, parent-child communication using Input and Output decorators, services, dependency injection, and introductory state management patterns.

Week 10: Advanced Angular Development

Week 11: Angular Testing

Week 17: Technical Interview Preparation
