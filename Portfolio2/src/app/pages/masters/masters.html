Academic Projects

Applications Programming Class

For the final Project for this course I developed a 3D space game using Python, OpenGL, and Blender. The project features player-controlled spacecraft navigation in a fully rendered 3D environment, with custom 3D models created in Blender and integrated into the game engine. Implemented collision detection for interactions between the player, projectiles, asteroids, and collectible objects, creating engaging gameplay mechanics. This project strengthened my understanding of game development concepts including object-oriented programming, real-time rendering, 3D transformations, collision systems, and asset integration.
